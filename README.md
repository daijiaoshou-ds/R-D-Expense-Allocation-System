# 🏦 研发分摊工具(测试版)

**做着玩的**

&gt; 把工资表、工时表、序时账扔进来，自动给你摊出一张审计底稿。支持"孤儿费用"轧差，自动试算平衡。

---

## ✨ 能干嘛？

| 功能 | 说明 |
|:---:|:---|
| 📊 **多赛道分摊** | 工资、社保、公积金分开算，各摊各的，不串味 |
| 🧮 **孤儿轧差** | 找到"有工资没工时"的憨批（比如离职补偿），按全员工时二次分摊 |
| 🧹 **序时账归集** | 直接费用自动归，间接费用按工时分，还能剔除重复科目 |
| ✏️ **在线改数** | 预览界面直接双击改Excel，不用来回倒腾文件 |

---

## 🧠 核心逻辑（简单说）
工资表（按人+月汇总） ──┐
├─> 按工时比例分摊 ──> 项目人工费明细
工时表（人+月+项目） ───┘                              
↓
序时账（直接+间接费用）────> 最终研发费用分摊表


**孤儿费用是啥？**
有些人 11 月发了工资但 10 月就离职了（11 月没工时），这笔钱如果不摊出去，试算不平。工具会自动识别这种"孤儿"，按**全期总工时**平摊到所有项目上。

---

## 🚀 三步上手

**1️⃣ 传文件**（左边栏）
- 工时表：姓名、工号、月份、项目号、工时
- 序时账：月份、项目号、科目、金额  
- 工资表（可选）：只填你有的字段，空着的工具会自动忽略

**2️⃣ 配字段**（中间页）
- **阶段一**：把 Excel 列名映射到标准字段（🔴必填 ⚪选填）
- **阶段二**：挑一下序时账里要剔除的科目（比如"应付职工薪酬"，避免和工资表重复）

**3️⃣ 出结果**（右边页）
点"开始分摊计算" → 检查试算平衡 → 下载 `审计底稿V9.xlsx`

---

## ⚠️ 使用须知

1. **轧差模式建议开启**，能把对不上的零钱自动摊平，试算差几分钱再也不用愁
2. **数据修改**：在"数据预览"页可以双击改数，改完记得点"保存修改"
3. **算完请人工抽查一两行**，被经理骂了别找我 🐶
